# Quito Fest 2025 – Web Version by Leandro Rubio

Este proyecto es una versión personalizada de la página web del Quito Fest 2025, desarrollada con React. Incluye secciones informativas sobre el evento, bandas participantes, horarios, galería, y más. El objetivo es brindar una experiencia moderna, visual y responsiva para los usuarios.

---

## Tecnologías Utilizadas
- React + Vite
- Tailwind CSS
- React Hooks
- Componentes reutilizables
- Diseño responsivo

---


## Características Principales
- Página totalmente responsiva
- Información del evento organizada en secciones
- Lista de bandas con tarjetas dinámicas
- Horario del evento por días
- Navegación fluida con componentes claros
